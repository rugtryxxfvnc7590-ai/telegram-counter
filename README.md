# Telegram 互推链接计数机器人

每天北京时间 0 点自动重置计数。

## 功能

- 监听指定 Telegram 群内的 X/Twitter 链接
- **收录**每条链接对应的 X 用户名与发送人电报 @（写入 `link_registry.json`）
- 群内满 40 条时自动提醒（机器猫人设）
- **不再**每条链接私信推送给主人
- 与本地「互推检查脚本」联动：检查到漏推时，报告里附带「他的电报号是 @xxx」

## 监听的群

- `TELEGRAM_CHAT_ID`（GitHub Secret，可逗号分隔多个）
- 固定额外群：`-1003891628675`

## 与互推检查脚本联动

收录数据保存在仓库根目录 `link_registry.json`，互推检查脚本会自动从 GitHub 拉取当日数据，在漏推报告里附上对应电报账号。
